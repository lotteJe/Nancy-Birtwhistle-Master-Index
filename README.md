# React App

Dit project is gebouwd met React en Vite.

## Lokale Ontwikkeling

1.  Installeer dependencies: `npm install`
2.  Start de dev server: `npm run dev`

## Deployment naar GitHub Pages

Dit project is geconfigureerd om automatisch te worden gedeployed naar GitHub Pages via GitHub Actions.

Zie [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md) voor gedetailleerde instructies over het opzetten van de GitHub repository en het configureren van de API keys.
